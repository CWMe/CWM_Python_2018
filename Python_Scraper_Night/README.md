# BeautfulSoup
## Install
```
pip install beautifulsoup4
```

## Run 1st example
```
python beautifulsoup-example.py
```

## Run 2nd example
```
python beautifulsoup-book.py
```

# Scrapy
## Install
```
pip install scrapy
```

## Run 1st example
```
scrapy runspider scrapy-simple.py -o - --output-format=json
```
## Run 2nd example
```
scrapy runspider scrapy-quotes.py -o - --output-format=json
```
